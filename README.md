# AWS S3 parallel download 

<https://blog.duyidong.com/2021/08/07/speed-up-s3-download/>

# How to use

### Environment

- JDK8/JDK11

Make sure you have AWS credentials in your environment parameters:

```bash
AWS_ACCESS_KEY_ID
AWS_SECRET_ACCESS_KEY
```

### How to run

```bash
./gradlew clean build
./gradlew run
```
